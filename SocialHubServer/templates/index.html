Social Media Poster Information

POST to Reddit

Endpoint: /postReddit

Type: POST

Input:

        {
            "title": "Your Title",
            "content": "Your Content",
            "client_id": "Your Client ID",
            "client_secret": "Your Client Secret",
            "password": "Your Password",
            "user_agent": "Your User Agent",
            "username": "Your Username",
            "community": "Your Community",
            "images": [file1, file2, ...]
        }

Use only content or images. If you use both the images will be the only ones published.

POST to Discord

Endpoint: /postDiscord

Type: POST

Input:

        {
            "url": "Your Discord Webhook URL",
            "title": "Your Text",
            "images": [file1, file2, ...]
        }

POST to Slack

Endpoint: /postSlack

Type: POST

Input:

        {
            "url": "Your Slack Webhook URL",
            "title": "Your Text"
        }
